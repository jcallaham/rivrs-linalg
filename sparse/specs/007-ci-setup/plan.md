## Implementation Design

### New Job: bench-sparse

Add a `bench-sparse` job to `.github/workflows/ci.yml` that:

1. Checks out the repository
2. Installs stable Rust toolchain
3. Configures build cache
4. Runs `cargo bench --no-run` in the `sparse/` directory

This verifies that the `solver_benchmarks` criterion binary (and any future bench targets) compiles without actually executing benchmarks.

**Job placement**: After the existing `doc-sparse` job, before any future jobs. Follows the existing naming convention (`{purpose}-sparse`).

**Toolchain**: Stable only (no MSRV matrix). Benchmarks are a development tool, not a library consumer concern. If benchmarks compile on stable, MSRV compilation adds no value.

**Cache**: Uses `Swatinem/rust-cache@v2`, consistent with other jobs.
